关于我的介绍

👨‍💻 个人简介

我是张良，一名拥有扎实数理素养的物理与计算科学探索者。我专注于偏微分方程的解析推导与数值计算，并对量子器件及材料的数值模拟怀有浓厚的兴趣。在实践方面，我具备良好的编程能力，能熟练运用 MATLAB 和 Python 进行科学计算与建模；同时拥有较好的英语学术交流与文献阅读能力。

🎓 教育背景

  • 中南林业科技大学 | 应用物理学（光电方向）| 本科
    • 学业表现：核心专业课平均分 90+，具备扎实的数学与物理理论基础。

🔬 科研经历

1. 低维材料热电输运与转化性质研究（第一性原理计算）

  • 核心方法：密度泛函理论（DFT）、非稳态格林函数（NEGF）、朗道输运理论。
  • 主要工作：利用 Quantum ATK 等仿真工具，对特定低维纳米带材料的晶格结构、电子能带以及热电输运性质进行系统性建模与计算。
  • 主要成果：成功解析了特定结构下的热电转换效率优化机制。相关研究成果已作为第一作者发表于物理学专业期刊 《Chinese Physics Letters》(CPL)。

🏆 竞赛与项目经历

1. 基于 RK3588 与 YOLO 的多任务能量标签智能检测系统

  • 项目角色：项目计划书写作手
  • 技术栈：OpenHarmony, RK3588 开发板, YOLO 目标检测算法
  • 项目描述：面向工业 AI 场景，在 RK3588 硬件平台上部署优化后的 YOLO 深度学习模型，实现对产品能量标签及外观缺陷的高精度实时多任务检测。主导了系统架构设计、算法端侧部署优化及团队协同。

2. 数学建模与算法竞赛

  • 全国大学生数学竞赛：荣获湖南赛区二等奖（前 9%）。
  • 数学建模竞赛实践：通过建立时间连续模型预测锂离子电池电量（SOC） 随时间演化规律，从最简单的情况出发，基于能量守恒原理和安时计量法导出时间连续方程；利用MATLAB实现卡尔曼滤波方法和扩展卡尔曼滤波法对SOC，和经验参数进行估计； LaTeX 规范排版并输出高质量学术论文。

🛠️ 专业技能

  • 物理/数据仿真：Quantum ATK (DFT 计算) / MATLAB (仿真计算) / Origin (数据可视化与分析)
  • 工程/算法栈：Python
  • 生产力工具：LaTeX 科技论文排版 / Git 版本控制
