hexo 搭建个人博客教程与计算物理核心代码库

发表于2026-05-18|更新于2026-05-18|教程代码

本文分为两个主要部分。第一部分整理并总结了在 Hexo 个人博客搭建与日常维护过程中最常用的控制台指令；第二部分则汇总了我在大学“计算物理训练”课程中编写的核心 MATLAB 代码片段，包括数值积分、非线性方程组求解、常微分方程（ODE）数值解以及数据插值拟合等关键算法实现。

第一部分：Hexo 博客搭建常用指令总结

在 Hexo 的日常使用中，绝大多数操作都可以通过 hexo 命令行工具完成。以下是博主最高频使用的指令汇总。

1. 基础构建与服务器运行

指令 简写 功能描述
hexo clean hexo c 清除缓存：删除之前生成的静态网页文件（public 文件夹）和缓存文件（db.json）。在更换主题、修改全局配置文件后必须执行。
hexo generate hexo g 生成静态文件：将 Markdown 源文件编译生成 HTML 静态网页，输出到 public 文件夹中。
hexo server hexo s 启动本地服务器：在本地 localhost:4000 启动服务器，用于本地实时预览博客的修改效果。
hexo deploy hexo d 自动部署：根据 _config.yml 中的配置，将生成的网站一键推送到 GitHub Pages 或云服务器。

第二部分：计算物理训练核心代码库（MATLAB篇）

在计算物理的训练中，运用 MATLAB 进行科学计算、数值模拟与数据处理是至关重要的。以下汇总了常用物理与数学问题的核心代码框架与解决方案。

1. 数值积分计算

利用数值方法求解连续积分。当精度要求不苛刻时，可通过离散求和的方式近似计算；对精度要求较高时，使用内置的高精度求积函数 quadl 或 integral。

2. 非线性方程组求解

计算物理中常遇到无法解析求解的复杂方程组。我们可以通过自编函数结合 fsolve 求数值解，或者利用 solve 进行符号求解。

3. 常微分方程（ODE）求解与矢量场相轨迹

物理体系中的动力学演化常由常微分方程（ODE）控制。ode45 是基于 4/5 阶龙格-库塔法的优秀求解器。同时，绘制解析解与数值解的对比，以及系统的相空间矢量场，是直观理解物理系统的有效手段。

4. 实验数据插值与最小二乘拟合

在处理物理实验数据或离散数值模拟数据时，我们需要避免“龙格现象”并对离散数据点进行平滑处理、插值扩充或趋势提取。

5. 高效计算：矩阵优化与预处理技巧

在 MATLAB 中编写大规模时域循环（如 FDTD、有限差分法 FDM）或处理巨型矩阵时，预分配内存与矩阵化（矢量化）操作能极大幅度提高代码的运行速度。

第三部分：计算物理训练核心代码库（Python篇）

除了 MATLAB，我们在训练中也使用 Python 作为重要的科学计算与数据处理工具。以下是我根据集训前五天内容梳理出的三个核心板块代码总结。

1. 数据清洗与 NumPy 数组基础 (Day 1 - Day 2)

本部分主要涉及 Python 基础数据结构（列表、字典）的操作、列表推导式的使用，以及利用 NumPy 进行高效的数据转换与统计计算。

2. 物理方程数值解与切片求导作图 (Day 3 - Day 4)

本部分展示了如何使用 Python 解决计算物理中的核心问题，例如通过显式与隐式欧拉法解刚性常微分方程（ODE），以及利用 NumPy 数组切片技巧替代 for 循环实现高效的数值导数计算，并使用 Matplotlib 绘制对比图。

3. 网络爬虫与数据归档 (Day 5)

除了物理计算，数据获取能力也必不可少。利用 requests 与 BeautifulSoup 可自动化抓取网页数据，并通过 csv 模块实现持久化归档。
